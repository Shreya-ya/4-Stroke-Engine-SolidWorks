# 4-Stroke-Engine-SolidWorks 
### SolidWorks CAD Design & Assembly

## 📌 Project Description

This project showcases the SolidWorks based design, modeling, and assembly of a four-stroke internal combustion engine.

By precisely designing the main load bearing and motion transmitting components, the project aims to comprehend engine kinematics.

The model shows how the crankshaft mechanism transforms the piston's reciprocating motion into rotary motion.

---

## 🎯 Project Objectives
The objective of the project :

- Design components of a 4-stroke engine
- Assemble components with correct constraints and alignment
- Use motion study to illustrate the fundamentals of engine operation
- Recognize how individual components cooperate to create fluid, continuous motion
  
---

## ⚙️ Components Modeled
The following components were individually designed and later assembled together:

- **Piston** – Transforms combustion pressure from combustion into linear motion  
- **Connecting Rod** – Transfers motion from piston to crankshaft  
- **Crankshaft** – Transforms reciprocating motion into rotational motion  
- **Crankshaft Pin** – Provides joint connection between crankshaft and the connecting rod  
- **Upper Block** –  Contains the piston which aids in guiding its motion 
- **Lower Block** – Completes the engine structure and supports the crankshaft

---

## 🧩 Assembly Details
All the parts were assembled using mates and constraints so that the engine moves in a realistic way.
Proper spacing, alignment, and smooth rotation were taken care of to make sure the mechanism runs without issues during the motion study.

## 🔄 Working Principle
The engine follows the standard **four-stroke operating cycle principle**:

- Intake Stroke: The piston descends during the intake stroke, allowing the fuel-air mixture to enter

- Compression Stroke: The mixture is compressed as the piston rises

- Power Stroke: The piston is pushed downward by combustion

- Exhaust Stroke: The exhaust gases are released during the exhaust stroke when the piston moves upward

- Continuous rotational output is guaranteed by the crankshaft-connecting rod mechanism
